金泰丽舍是北京品质国企——金泰地产集团联合北京东管头投资管理公司在武清开发的鼎力之作，也是武清面向北京地区的温泉休闲综合体大盘。项目位于京津城际高铁武清站东南500米，华北地区的奢侈品折扣店——佛罗伦萨小镇南侧，紧邻龙凤河、京杭大运河及永定河，为三江汇流天然氧吧之地。 金泰丽舍是区域内实现户户温泉直入的品质大盘。项目占地350亩，规划总建筑面积约48万平方米，容积率仅为1.8，整体绿化率达到40%以上。住宅产品类型包括四层品质叠墅、六层花园电梯洋房、精装河景高层等，面积在58-150平方米之间；自身配套商业包括管家式私属温泉会所、高端精品温泉酒店、北京品牌幼教机构、老年活动中心、业主食堂、果蔬直供基地、城市运动公园等。项目整体分为07、08、09三个组团三期进行开发，预计2016年全部建成。 金泰丽舍整体定位为“地中海水岸温泉休闲小镇”，整体选用新古典主义建筑形式，色调运用轻松、明快，在富有现代感的简约线条之下又融入了武清作为千年运河古镇的优雅与内涵。 金泰丽舍地处武清城市核心位置，周边配套齐全、交通发达。项目距北京五环70公里，天津外环仅10公里，距京津城际武清站仅500米，京津塘、京津、京沪、津保等5条高速以及103、104、112等国道均可直达本项目。周边城市医疗、教育、商业、休闲娱乐等配套设施均在15车程以内。如中国华北地区规模、种类齐全的名品折扣中心——佛罗伦萨小镇，5即达；我国北方规模的儿童主题游乐园——凯旋王国，8即达；全国著名高中，教育质量超群的杨村一中近在咫尺。 金泰丽舍具有得天独厚的水岸生态资源，南侧的龙凤河与项目咫尺之间，与东侧的京杭大运河、永定河共同构筑了本项目的湿地自然景观，加之贯穿项目南北的城市绿地公园，使本项目宛若置身天然绿肺氧吧，充分满足现代都市人“静享水岸慢生活”的心灵渴求。 金泰丽舍倾力打造的武清高品质养生温泉会所，总建面近10000㎡，会所采用会员制的运营管理模式，为客户提供高品质、私属化的定制化产品及管家式服务。差异化的产品设置、纯粹的中医养生主题温泉理疗及高端私属化的定制服务是本项目的核心特色，满足都市快节奏精英人群舒缓压力、调节身心、养生保健以及私属商务的全方位需求； 　　金泰丽舍是区域内采用互动参与式景观园林打造的住宅项目，在重点突出内外水景结合以及生态环境自然融合的基础上，以业主认养主题DIY果园的形式让都市业主参与其中，配合儿童木船游乐场、细润白沙坑、老年养生广场、河岸健步慢跑道、温泉戏水池等，为北京及周边养生、度假人群打造理想的都市生态休闲居所。
